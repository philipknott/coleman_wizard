Coleman Wizard

Press Option + Command + i to open developer console.


createEvent(day, time, command)

Creates an event with the given arguments in the Coleman calendar.

day: A number from 0-6 which corresponds to a weekday (Sunday to Saturday, respectively).

time: A string in HH:MM:SS format (e.g. 07:30:00 for 7AM).

command: A string with either a color or the word 'off'

clearAllEvents()

Deletes all events under the Coleman calendar.
Takes no arguments.
